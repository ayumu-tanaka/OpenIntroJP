# 貢献者行動規範

本プロジェクトの貢献者およびメンテナーとして、私たちは、問題の報告、機能の要望、ドキュメントの更新、プルリクエストやパッチの提出、その他の活動を通じて貢献するすべての人々を尊重することを誓います。

私たちは、経験レベル、性別、性自認および表現、性的指向、障がい、外見、身体の大きさ、人種、民族、年齢、宗教にかかわらず、すべての人がハラスメントのない形で本プロジェクトに参加できる環境を作ることに尽力します。

参加者による容認できない行動の例には、性的な言葉や画像の使用、中傷的なコメントや個人攻撃、荒らし行為、公的または私的なハラスメント、侮辱、その他プロフェッショナルにふさわしくない行為が含まれます。

プロジェクトのメンテナーは、本行動規範に沿わないコメント、コミット、コード、Wikiの編集、Issue、その他の貢献を削除、編集、または拒否する権利と責任を有します。本行動規範に従わないプロジェクトのメンテナーは、プロジェクトチームから除名される場合があります。

虐待的、ハラスメント的、またはその他の容認できない行為の事例は、Issueを立てるか、プロジェクトのメンテナーのいずれかまたは複数人に連絡することで報告できます。

この行動規範は、[Contributor Covenant](http://contributor-covenant.org) バージョン 1.0.0 を採用しており、[こちら](http://contributor-covenant.org/version/1/0/0/)で入手可能です。
